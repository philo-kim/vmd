# Support

VMD is an experimental public draft.

Use GitHub Issues for:

- syntax proposals
- renderer bugs
- Chrome extension bugs
- VS Code extension bugs
- documentation gaps
- sample document proposals

Before opening an issue, include:

- the `.vmd` source or a minimal reproduction
- the renderer or extension used
- your browser or VS Code version, if relevant
- expected behavior
- actual behavior

For security-sensitive reports, follow `SECURITY.md`.
